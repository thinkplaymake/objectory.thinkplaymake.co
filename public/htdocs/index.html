objectory

What is objectory?

objectory is a platform and factory for moving object projects. moving object projects are where users can release and find objects which move around the world, attaching stories to them as they go - think bookcrossing.org and disposablememoryproject.org

we're building the platform as we go, adding new functionality to react to our needs, with the aim of eventually converting the disposable memory project to use objectory, as well as allowing other developers and users to create their own similar projects.

the code will be available at all times via github, and we'll be taking public feedback on board always. to keep posted, subscribe to the newsletter.

_

objectory is a portmanteau of the words 'object', 'trajectory', 'factory' and 'story', the four cornerstones of our project:

  1. object: a physical thing, which can be picked up, touched, smelt, felt, tasted, handed over, lost. this is about physical objects that cannot be infinitely replicated.
  2. trajectory: the path in which something travels. this is about movement, and passing from person to person, often in random directions.
  3. factory: a place where many of the same thing are made. this is about enabling as many object based story telling projects to exist as possible, not just by those who can, but by those who want to.
  4. story: a narrative. the connection between our objects, time, people and places.
